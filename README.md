# 学習塾 LP（90HomePage）

高級感・モダン・ミニマルをコンセプトにした学習塾のランディングページです。

## 技術スタック

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React

## デザイン

- **メインカラー:** `#31A9C3`
- **アクセントカラー:** `#20284E`
- **フォント:** Noto Sans JP（本文）、Noto Serif JP（見出し）

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## Hero Section の背景画像

ヒーローセクションに背景画像を表示する場合は、**`public/hero-bg.jpg`** に画像を配置してください。  
画像がない場合は、アクセントカラー系のグラデーションが表示されます。
